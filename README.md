Cybersecurity Awareness Chatbot GUI

This is a C# Windows Forms GUI application for the final cybersecurity chatbot task.

Features

- GUI-based chatbot using Windows Forms controls.
- Task assistant for adding, viewing, completing, and deleting cybersecurity tasks.
- Optional reminders stored with tasks.
- MySQL storage for task data.
- Cybersecurity mini-game quiz with more than 10 questions.
- Immediate quiz feedback and final score.
- NLP simulation using keyword detection and regular expressions.
- Activity log showing recent task, reminder, quiz, and NLP actions.

Example Chat Commands

- Add task - Review privacy settings
- Yes, remind me in 3 days
- Remind me to update my password tomorrow
- Show activity log
- Start quiz
- Show tasks
